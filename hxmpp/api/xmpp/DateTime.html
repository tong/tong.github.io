class xmpp.DateTime

Available on all platforms

Standardization of ISO 8601 profiles and their lexical representation.

XMPP Date and Time Profiles: http://xmpp.org/extensions/xep-0082.html

The Date profile defines a date without including the time of day. CCYY-MM-DD

The DateTime profile is used to specify a non-recurring moment in time to an accuracy of seconds (or, optionally, fractions of a second). CCYY-MM-DDThh:mm:ss[.sss]TZD

The Time profile is used to specify an instant of time that recurs (e.g., every day). hh:mm:ss[.sss][TZD]
