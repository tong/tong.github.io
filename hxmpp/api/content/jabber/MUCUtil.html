class jabber.MUCUtil
Static methods for parsing of mutliuser chat addresses.
static var EREG : EReg
Regular expression matching a valid MUC address
static inline function getHost(t : String) : String
Returns the host of the muc jid.
static function getNick(t : String) : String
Returns the occupant name of the muc jid.
static function getParts(t : String) : Array<String>
Returns array existing of roomname0, host1 and (optional) occupantname2 of the given muc address.
static inline function getRoom(t : String) : String
Returns the room of the muc jid.
static function isValid(t : String, ?nick : Bool) : Bool
Returns true if the given string is a valid muchat address. The 'nick' parameter indicates if a full adress (including nickname) is expected.
