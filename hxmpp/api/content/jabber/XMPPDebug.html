class jabber.XMPPDebug

Utility for debugging XMPP transfer. Set the compiler flag: -D xmpp_debug to activate it.

Terminal targets: Color highlighted Browser targets: Printed to the default debug console Adobe air: Printed to fdb 'trace'

static var active : Bool
Activate/Deactivate output. Default is true.
static var beautify : Bool
Indicates if the XMPP debug output should get formatted/beautified. Mind: If active, it is not ensured that the formatted string matches exactly the sent/recieved ones! Default value is false. Currently only supported in terminal targets.
