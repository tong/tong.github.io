class jabber.sasl.LOGINMechanism

LOGIN SASL Mechanism.

Obsolete clear-text user/password Simple Authentication and Security Layer (SASL) mechanism called the LOGIN mechanism. The LOGIN mechanism was intended to be used, in combination with data confidentiality services provided by a lower layer, in protocols which lack a simple password authentication command.

http://tools.ietf.org/id/draft-murchison-sasl-login-00.txt
