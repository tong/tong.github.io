class xmpp.EntityTime

var tzo : String
The entity's numeric time zone offset from UTC. The format conforms to the Time Zone Definition (TZD) specified in XEP-0082 (http://www.xmpp.org/extensions/xep-0082.html). Example: +02:00

The UTC time according to the responding entity. The format conforms to the dateTime profile specified in XEP-0082 (http://www.xmpp.org/extensions/xep-0082.html) and MUST be expressed in UTC.
